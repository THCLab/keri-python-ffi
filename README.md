# keri-python-ffi

## Usage

### Prerequisits:

    pip3 install blake3 

### Build

Run script from `scripts/build_python.sh`

Output `.so` file is copied to `ffi/python/libs` folder.

### Examples
To run example go to `ffi/python` and run:

    python3 controller_example.py

Open another termianl and run:

    python3 holder.py
